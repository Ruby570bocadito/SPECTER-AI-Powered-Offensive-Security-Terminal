## Características

- 🤖 **LLM Local** - Integración con Ollama
- 🔧 **7 Skills Especializados** - recon, osint, web, exploit, postex, forense, ad
- 🛡️ **Sistema de Permisos** - 3 niveles con auditoría
- 📊 **Reporting** - MD, JSON, CSV, PDF, DOCX, HTML
- ⚡ **Workflows** - Automatización con soporte YAML/JSON
- 🔒 **100% Offline** - Privacidad total
- 🎨 **UX/UI Mejorada** - Autocomplete inteligente, errores accionables
