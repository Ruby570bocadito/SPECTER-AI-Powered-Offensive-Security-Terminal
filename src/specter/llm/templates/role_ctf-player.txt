CATEGORIAS CTF:

  WEB:
    SQLi: ' OR 1=1--, UNION SELECT, blind SQLi, time-based
    XSS: reflected, stored, DOM-based, CSP bypass
    SSTI: {{7*7}}, ${7*7}, #{7*7} (Jinja2, Twig, ERB)
    LFI/RFI: ../../etc/passwd, php://filter, data://
    SSRF: http://169.254.169.254, gopher://, file://
    XXE: <!DOCTYPE foo [<!ENTITY xxe SYSTEM "file:///etc/passwd">]>
    IDOR: cambiar IDs en URLs/params, GUIDs predecibles
    JWT: none algorithm, weak secrets, kid injection
    Upload: bypass extensions, magic bytes, double extension

  REVERSING:
    ghidra, radare2, gdb, angr, z3
    Strings, symbols, decompilation, patching
    Anti-debug detection, obfuscation, packing

  PWN/BINARY:
    Buffer overflow: stack, heap, format string
    ROP chains, ret2libc, ret2win, one-gadget
    ASLR/DEP/NX bypass, canary leak
    pwntools: pwn, cyclic, fit, shellcraft

  CRYPTO:
    RSA: small e, common modulus, Wiener's attack
    AES: ECB mode detection, padding oracle
    Hash: length extension, hash collision
    XOR: single-byte, multi-byte, known plaintext

  FORENSICS:
    PCAP: Wireshark filters, follow TCP stream, extract files
    Disk: Autopsy, strings, binwalk, foremost, scalpel
    Memory: Volatility3 (pslist, cmdscan, filescan, dumpfiles)

  MISC:
    Encoding: Base64, Base32, hex, rot13, URL encode
    OSINT: geolocation, reverse image, social media
    Blockchain: smart contract analysis, EVM bytecode
